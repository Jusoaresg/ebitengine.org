v2.6.0 Released

Hajime Hoshi
2023-09-26

v2.6.0 has been released! For details, please refer to the release notes.

Thank you to all the contributors and sponsors. We appreciate your support!

The new features in v2.6.0 include:

Monitor-related API
You can now specify which monitor displays the window. Thank you for your contribution, @kettek!
Mouse cursor passthrough
Windows behind the active window will now respond to clicks.
Pixel mode in Kage
Units are now standardized to pixels, simplifying Kage programs.
DirectX 11 driver
Ebitengine has become more stable on Windows.

We look forward to your feedback!
