v2.6.0 がリリースされました! 詳しくはリリースノートを参照してください。

すべてのコントリビューターとスポンサーの皆様に感謝いたします。どうもありがとうございます!

v2.6.0 に入った新しい機能は次のとおりです:

モニター関係の API
ウィンドウを表示するモニターを指定できます。 @kettek さん、コントリビュートありがとうございます!
マウスカーソルのパススルー
(ウィンドウの背後にあるウィンドウがクリックに反応するようになります。
Kage のピクセルモード
単位がピクセルに統一され、 Kage プログラムが簡単になります。
DirectX 11 ドライバ
Windows でより安定するようになりました。
