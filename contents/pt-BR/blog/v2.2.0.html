v2.2.0 is released

Hajime Hoshi
2021-10-05

I am very happy to announce the release of v2.2.0 (Release Note).

I appreciate all the contributors and all the sponsors. Thank you very much!

v2.2.0 includes these new features:

  • Supporting Nintendo Switch™
  • New audio implementation
  • Standard gamepad layout mappings
  • New vector graphics APIs
  • macOS's native fullscreen
  • etc.

v2.2.0 also improves performance in terms of speed and memory usage.

I appreciate your feedbacks!
