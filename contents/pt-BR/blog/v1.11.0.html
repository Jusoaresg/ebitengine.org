v1.11.0 is released

Hajime Hoshi
2020-04-13

We are very happy to announce the release of v1.11.0 (Release Note). The new features are:

  • Game interface and RunGame
  • Resizable window
  • Window position
  • Transparent screen
  • Floating window
  • Capturing cursor

You can develop a desktop mascot application with these API. See the mascot example.

Enjoy!
